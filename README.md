# Lahiq App (لحّق) ⚡️

**Lahiq** is a fast-paced Arabic word-challenge game built with **Flutter**. It tests players' quick thinking by requiring them to name a specific number of items starting with a random letter within a 7-second countdown.

## ✨ Key Features
* **Dynamic Challenge Engine**: Automatically generates random challenges combining Arabic letters and various categories (e.g., Restaurants, Cities, Brands).
* **Real-time Interaction**: Features a custom tap-to-count system where players log their answers by tapping the screen.
* **Visual Feedback**: Includes a reactive timer that changes color from green to red as time runs out, providing immediate visual urgency.
* **Modern UI/UX**: Designed with a sleek dark theme, utilizing `RadialGradient` and `LinearGradient` for a polished, professional look.
* **Arabic Typography**: Fully optimized for Arabic with the **Tajawal** font integrated via Google Fonts.

## 🛠 Tech Stack
* **Framework**: Flutter.
* **Language**: Dart.
* **UI Components**: Material 3.
* **State Management**: `StatefulWidget` with custom timers.
* **Animations**: `AnimationController` for dynamic background backdrops.

## 🚀 How to Run
1. Ensure you have the Flutter SDK installed.
2. Clone the repository and navigate to the project folder.
3. Run `flutter pub get` to install dependencies.
4. Launch the app using `flutter run`.
